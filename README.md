# Development
#### Install dependencies
```sh
$ git clone https://github.com/Muhammad-Sarfaraz/Bonfire.git
$ cd Bonfire && npm install
```

```sh
# Run the project
$ npm run start
```

```sh
# Build the project
$ npm run build
```
